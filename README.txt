Для начала работы необходимо создать или импортировать базу данных. Для этого можно запустить на исполнение файл "create_db_and_table.php" 
перейдя по ссылке http://"название-вашего-хоста"/php/create_db_and_table.php
или импортировать файл history.sql непосредственно в СУБД
или создать базу данных и таблицу вручную (название базы данных "test_problem", название таблицы "history"(первый столбец "id" INT авто_инкремент,
второй столбец "string" TEXT)

Версии которые я использовал:

PHP - 8.0
MYSQL - 8.0
